## 1. Product Overview  
The **NH-Bulb Glow RGB+** is a Wi-Fi-enabled smart LED bulb that allows full **color customization, scheduling, dimming, and automation** through the NeuraHome App.  
Key features:  
- 16 million colors + adjustable white temperature  
- Energy-efficient LED  
- Voice control (Alexa / Google Home / Siri)  
- Automation & scheduling  
- Remote control via app  
- Grouping multiple bulbs

## 4. Installation & Setup

### Step 1 — Install the Bulb
- Screw into standard E26/E27 socket  
- Ensure power is off before installation

### Step 2 — Power On
- Switch on the wall power  
- Bulb will blink **white** indicating pairing mode

### Step 3 — Pair via App
1. Open NeuraHome App → **Add Device → Smart Bulb**  
2. Select **NH-Bulb Glow RGB+**  
3. Ensure phone is connected to 2.4 GHz Wi-Fi  
4. Follow on-screen instructions  

### Step 4 — Optional Grouping
- Add multiple bulbs to a **room or group** for synchronized control

## 6. LED Indicator Guide
| Status | Meaning |
|--------|---------|
| Blinking White | Ready for pairing |
| Solid White | Connected to Wi-Fi |
| Blinking Blue | Firmware update in progress |
| Blinking Red | Connection error |
| No light | Power off or bulb faulty |

## 7. Scheduling & Automation
- **Time-based scheduling:** Turn on/off at set times  
- **Sunrise/sunset mode:** Automatically adjusts brightness  
- **Event triggers:** Link with motion sensors or cameras for smart automation  
- **Group automation:** Control all bulbs in a room simultaneously  

---

## 8. Firmware Updates
- Automatic by default  
- Manual update via App → Device Settings → Firmware Update  
- Ensure Wi-Fi is stable  
- Avoid power interruption during update  

---

## 9. Troubleshooting
### Bulb not connecting
- Check 2.4 GHz Wi-Fi  
- Ensure bulb in pairing mode  
- Reset bulb (power off/on 3 times)

### Bulb flickering
- Ensure bulb screwed in tightly  
- Check voltage compatibility  
- Verify app brightness level

### Remote control not working
- Ensure app updated to latest version  
- Check phone Wi-Fi connection  
- Remove & re-add bulb in app if needed

---

## 10. Error Codes
| Code | Meaning | Action |
|------|---------|--------|
| B101 | Wi-Fi not detected | Reconnect to 2.4GHz network |
| B202 | Firmware update failed | Retry in app |
| B303 | Power fluctuation detected | Check socket |
| B404 | Bulb not responding | Power cycle bulb |
| B505 | Group sync failed | Refresh group in app |
