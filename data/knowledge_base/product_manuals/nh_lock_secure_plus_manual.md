## 4. Installation & Setup  
### Tools Required  
- Screwdriver  
- Drill (only if old lock holes differ)

### Steps  
1. Remove the existing door lock hardware.  
2. Install the exterior keypad module.  
3. Place the interior mounting plate and secure with provided screws.  
4. Connect the cable between the keypad and interior module.  
5. Insert the battery pack and secure the interior cover.  
6. Power on the device — the keypad lights will flash **blue**.  
7. Proceed with app pairing.

---

## 5. Mobile App Pairing  
### Steps  
1. Open **NeuraHome App → Add Device → Smart Lock**.  
2. Enable **Bluetooth & Location** on your phone.  
3. Press and hold the `#` key for 3 seconds until the keypad flashes **blue**.  
4. The app will discover `NH-Lock Secure+`.  
5. Select your Wi-Fi network and enter the password.  
6. Wait for configuration (10–15 seconds).  
7. App will show “Device Connected”.

## 6. LED & Keypad Indicator Guide  
| Indicator | Meaning |
|----------|---------|
| Blue (slow blink) | Ready for pairing |
| Blue (solid) | Successfully connected |
| Green | Correct PIN / fingerprint accepted |
| Red | Incorrect PIN / fingerprint |
| Red (fast blink) | Low battery |
| Yellow | System configuration or firmware update |

## 7. Device Controls  
### Unlock Methods  
- **PIN Unlock:**  
  Enter PIN → press `#`.  
- **Fingerprint Unlock:**  
  Place finger on scanner for 1 sec.  
- **App Remote Unlock:**  
  Tap **Unlock** in app → Cloud verifies → Door opens.  
- **Backup Physical Key:**  
  Hidden key slot under top cover.

### Lock Methods  
- Auto-lock after **5–30 seconds** (configurable).  
- Manual lock: press and hold `*` for 2 seconds.  
- Lock via app: tap **Lock**.

---

## 8. Temporary Guest PINs  
### Create Guest PIN  
1. App → Smart Lock → Guest Access.  
2. Select validity:  
   - One-time  
   - Time-limited  
   - Recurring  
3. Share PIN via WhatsApp/SMS/email.

### Revoke Guest PIN  
App → Guest Access → Select PIN → Delete.

---

## 9. Firmware Updates  
1. App → Device Settings → Firmware Update.  
2. Ensure lock battery > 40%.  
3. Update takes 2–3 minutes.  
4. Lock LEDs show **yellow** during update.

## 11. Error Codes  
| Code | Meaning | Action |
|------|---------|--------|
| E01 | Fingerprint sensor error | Clean sensor, retry |
| E02 | Motor jam | Check alignment |
| E03 | Network connection failed | Reconnect Wi-Fi |
| E04 | Low battery | Replace batteries |
| E05 | Incorrect PIN attempts exceeded | Lockdown for 60 sec |
| E06 | Firmware update failed | Retry in app |

## 12. Troubleshooting  
### Lock not responding  
- Remove and reinsert batteries.  
- Check that ribbon cable is properly connected.

### Bluetooth not connecting  
- Move phone closer to lock (< 1 meter).  
- Restart the app.

### Wi-Fi keeps disconnecting  
- Ensure door area has strong 2.4 GHz signal.  
- Avoid metal doors blocking signals.  
- Reset Wi-Fi settings in app.

---

## 13. Safety & Maintenance  
- Do not expose to direct rain (use shield for outdoor installation).  
- Replace batteries every 6–10 months.  
- Clean keypad with soft cloth only.  
- Avoid storing backup keys near the door.  
- Keep firmware updated.
